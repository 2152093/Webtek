    • Hypertext Transfer Protocol

    1. HTTP 0.9 - 1991 on;y 3 methos (get, head, post)

    2. HTTP 1.0 (RFC 1945, May 1996) all 8 methods

    3. HTTP 1.1 (RFC 2068 Jan 1997, RFC 2616 June 1999)

    HTTP runs on top of TCP/IP using TCP port 80 by default or TCP port 443 for HTTPS (HTTP over SSL/TLS)

    HTTP is based on a client server architecture

    > Client (aka User Agents) i.e., web browsers, web crawlers, and other end user tools and applications

    > Servers i.e., origin servers, proxy servers, gateway

    HTTP uses a request-response standard protocol

    HTTP is a stateless communication protocol

    HTTP provides support for other functionalities
